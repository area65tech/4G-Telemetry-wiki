Troubleshooting
===============

Common Issues
-------------

+----------------------------------+----------------------------------+----------------------------------+
| Issue                            | Possible Cause                 | Solution                         |
+==================================+==================================+==================================+
| No network connection            | SIM not detected                 | Reinsert SIM, check data plan    |
+----------------------------------+----------------------------------+----------------------------------+
| No GPS signal                    | Antenna not connected            | Verify GPS SMA connection        |
+----------------------------------+----------------------------------+----------------------------------+
| No data in cloud                 | Network issue                    | Check 4G signal strength         |
+----------------------------------+----------------------------------+----------------------------------+
| Data not logging                 | MicroSD not detected             | Reinsert microSD card            |
+----------------------------------+----------------------------------+----------------------------------+
| Interface communication failure  | Wrong interface settings         | Verify interface configuration   |
+----------------------------------+----------------------------------+----------------------------------+
| Power issues                     | Insufficient current             | Use 5V @ 1A minimum supply       |
+----------------------------------+----------------------------------+----------------------------------+

Debug Checklist
---------------

1. Verify power supply (5V @ 1A)
2. Check SIM card insertion and status
3. Verify antenna connections (4G + GPS)
4. Confirm interface wiring and settings
5. Check microSD card insertion
6. Review cloud platform device registration
7. Monitor LED indicators for status

LED Indicators
--------------

+--------+--------+----------------------------------+
| LED    | Status | Meaning                          |
+========+========+==================================+
| Power  | Solid  | Power OK                         |
+--------+--------+----------------------------------+
| 4G     | Blinking| Network activity                 |
+--------+--------+----------------------------------+
| GPS    | Solid  | GPS lock acquired                |
+--------+--------+----------------------------------+
